# CodeChallenge
The purpose of this application is to load a set of locations from a JSON file and display the locations as a pin on the map.

## Technologies used
- KotlinX Serialization
- Retrofit & OKHttp
- Kotlin Coroutines
- Live Data (AndroidX)
- ViewModel (AndroidX)
- GoogleMaps
- Leak Canary
- Dagger Hilt
